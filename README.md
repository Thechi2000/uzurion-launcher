# Tauri + Next.js + Typescript

## Development
Requirements:
- [npm](https://www.npmjs.com/) (through [node](https://nodejs.org/en/download/)) for NextJS
- [cargo](rustup.rs) for Rust

To run the app for dev, use
> `npm run tauri dev`
